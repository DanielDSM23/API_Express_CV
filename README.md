## Installation

1. Cloner le repo:
   ```bash
   git clone https://github.com/DanielDSM23/API_Express_CV.git
   ```
2. Naviguez dans le dossier :

 ```bash
   cd API_Express_CV/
   ```

3. Lancer le projet :

 ```bash
   node app.js
   ```
